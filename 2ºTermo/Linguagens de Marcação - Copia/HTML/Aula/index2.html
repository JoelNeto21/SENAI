O que é o Box Model?

O Box Model é o conceito que define como os elementos HTML são renderizados como caixas retangulares. Cada caixa é composta por:

  • Content (Conteúdo): O conteúdo do elemento (texto, imagem, etc).
  • Padding: Espaço interno entre o conteúdo e a borda.
  • Border: A borda ao redor do padding.
  • Margin: Espaço externo entre a borda e outros elementos.

O que são elementos Block Level e Inline?

Block Level: Elementos que ocupam toda a largura disponível e sempre começam em uma nova linha.

<div> (block)
<p> (block)
<h1> (block)

Inline: Elementos que ocupam apenas o espaço necessário e não quebram a linha.

<span> (inline) <a> (inline) <strong> (inline)
  • Block Level: Quebra linha, ocupa toda a largura.
  • Inline: Não quebra linha, ocupa só o necessário.
